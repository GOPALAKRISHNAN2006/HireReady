# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-30

### Added
- AI-powered mock interview system with OpenAI GPT-4 and Google Gemini
- Real-time feedback and response evaluation using NLP
- User authentication with JWT (register, login, password reset)
- Admin panel for user and question management
- Analytics dashboard with interactive charts
- Achievements and badge system
- Group discussion practice with AI moderators
- Communication assessment features
- Aptitude test module (quantitative, logical, verbal)
- Career roadmap and personalized recommendations
- AI chatbot assistant
- Question bank with bookmarking
- Company-specific preparation tips
- Docker support with docker-compose
- CI/CD pipeline with GitHub Actions
- E2E testing with Playwright
- Accessibility testing with jest-axe
- API documentation (OpenAPI/Swagger, Postman)
- Security scanning with CodeQL
- Automated dependency updates with Dependabot

### Security
- Helmet.js for HTTP security headers
- Rate limiting to prevent abuse
- bcrypt password hashing
- JWT token authentication
- CORS configuration
